# NEON FLOW - ONE LINE PUZZLE
Это медитативная головоломка, в которой нужно соединить все клетки поля одной непрерывной линией, огибая препятствия. Игра сочетает в себе расслабляющий геймплей с красивой эстетикой.

**Ключевые особенности:**
- 221 уровень с постепенным нарастанием сложности
- **8 тематических этапов** — от «Первых шагов» до «Легенды вечности»
- Тактильные элементы — бомбы для разрушения блоков, бонусы-подсказки, временные бонусы
- **Прогрессивная система** — чтобы открыть новый этап, нужно набрать достаточно звёзд

**Атмосфера:**
- **Тёплая неоновая графика** с плавными переливами цвета
- **Расслабляющая электронная музыка**, создающая медитативное настроение
- Приятные тактильные отклики и визуальные эффекты

Для тех, кто любит **успокаивающие головоломки**, красивый визуал и неторопливый, но увлекательный игровой процесс. Идеально, чтобы отвлечься и погрузиться в потоковое состояние.
